Lessons learned

Nesse projeto podemos desenvolver um site aplicando o conhecimento obtido nos primeiros três blocos do curso.
Buscando cumprir alguns requisitos de posicionamento de elementos, estilização e semantica.

Para mim o principal desafio, foi lidar com o requisito dois que tinha a especificação de manter a barra superior com o titulo fixo, o que inicialmente fez os demais conteúdos ficarem "aglomerados" no mesmo lugar. Resolvi esse problema utilizando display em block para alterar a posição de cada seção.
